# Xeneon Notes

Standalone widget in the same visual theme as the existing Xeneon widgets.

Features:
- Two switchable pages: Notes and Todo
- Local persistence with `localStorage`
- Minimal commands for quick use:
  - Notes: add, sample, pin latest, pin/unpin, delete, clear all
  - Todo: add, sample, toggle done, delete, focus mode, clear done

Open `notes.html` as a local widget page.
