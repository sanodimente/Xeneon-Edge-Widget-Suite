# Xeneon Notes

Standalone widget in the same visual theme as the existing Xeneon widgets.

## What It Does

- Provides two switchable pages: Notes and Todo.
- Stores data locally with `localStorage`.
- Works without WidgetBridge.

## Current Behavior

- Notes support inline add, edit, delete, and clear-all.
- Todo supports inline add, edit, done toggle, delete, and clear-all.
- Both views keep the same large-text layout and minimal footer actions.

## Using It In Corsair iCUE Xeneon Edge

1. Open `Notes/notes.html` in a text editor.
2. Copy the full file contents.
3. Open the iframe widget inside the Corsair iCUE app for Xeneon Edge.
4. Paste the copied HTML into the widget content field.

No local bridge process is required for this widget.
